# How to contribute to this repository?

## Adding new skills

We are only accepting skills that are convering Kotlin language itself or `kotlinx.` libraries. Before
submitting a skill, prefer opening a new issue and describing the skill you want to add. 
We will review it and provide feedback.

We will **not** accept skills that are using/adding some third-party dependencies. Exception is well-known and widely
used libraries.

### How to create a new skill

Check out the [skill-creator](https://github.com/anthropics/skills) skill.

## Adding a new skills distribution way

Prefer opening a new issue and describing the new distribution way you want to add. 
We will review it and provide feedback. Then you could submit a PR with the required changes.
