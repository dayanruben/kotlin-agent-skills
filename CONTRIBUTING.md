# How to contribute to this repository?

## Adding new skills

We are only accepting skills that are covering Kotlin language itself or `kotlinx.` libraries. Before
submitting a skill, prefer opening a new issue and describing the skill you want to add. 
We will review it and provide feedback.

We will **not** accept skills that are using/adding some third-party dependencies. Exception is well-known and widely
used libraries.

### How to create a new skill

Check out the [skill-creator](https://github.com/anthropics/skills/tree/main/skills/skill-creator) skill.

### Quick contribution checklist

- ✅ The skill must follow the [Agent Skills specification](https://agentskills.io/specification)
- ✅ Skill has working SKILL.md with YAML frontmatter where `name` and `description` are required fields
- ✅ Clear documentation and use cases
- ✅ Skill category is properly selected
- ✅ No security vulnerabilities or malicious code

## Adding a new skills distribution way

Prefer opening a new issue and describing the new distribution way you want to add. 
We will review it and provide feedback. Then you could submit a PR with the required changes.
